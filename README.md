<h3> Instructions <h3>

1. Clone me
`git clone git@github.com:leon-do/slackJS-questions.git`


2. These questions are listed as an array. To add a question, add a comma, make a new line, put the question in quotes and you're done.

3. Example:
```
<!--Before-->

[
    "question1",
    "question2"
]

<!--After-->
[
    "question1",
    "question2",
    "question3"
]
```

4. Pull request
